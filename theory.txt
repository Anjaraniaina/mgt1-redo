a.	Que permet de faire la commande git pull ? 
La commande git pull permet de mettre à jour le dépôt local, et synchronizer
le dépôt distant et le dépôt local.

b.	Que permet de faire la commande git clone ? 
La commande git clone permet de cloner ou obtenir un dépôt distant dans mon environnement local.

c.	Expliquer ce que fait un « fork » ?
Un fork permet d'avoir une copie d'un dépôt déjà existant.

d.	Expliquez brièvement l’utilité d’une pull request (PR) ? 
Une PR est une demande faite d'une branche A à une branche B cible pour que la 
branche cible B obtienne les derniers changements ou commits de branche A.
